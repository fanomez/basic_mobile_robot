## Get started with Gitbug CLI
- Run:  
``` 
gh auth login 
```

- Login parameters :
    - Account to log into : **Github.com**

    - Preferred protocol for Git operations : **SSH**

    - Generate a new SSH key to add to your Github account : **yes**

    - Title for your SSH key : **Github CLI**

    - How would you like to authenticate Github CLI : **Login with a web rowser**

## Install Joint state publisher package

``` 
sudo apt update
sudo apt install ros-jazzy-joint-state-publisher-gui
```

## Install ros2_control package
``` 
sudo apt update
sudo apt install ros-jazzy-ros2-control
sudo apt install ros-jazzy-ros2-controllers
```
- These install :
    - controller_manager
    - joint_state_broadcaster
    - diff_drive_controller
